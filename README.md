# Análise Exploratória de Dados - Netflix 📊

Este projeto consiste numa análise exploratória detalhada do catálogo da Netflix, desenvolvida em **Python** utilizando o ambiente Google Colab. O objetivo principal foi extrair insights sobre a distribuição de títulos, anos de lançamento e composição do elenco.

---

### 🚀 O Desafio Técnico
Ao contrário de análises convencionais que utilizam apenas a biblioteca Pandas, este projeto focou na **manipulação de estruturas nativas do Python**. 
- O dataset foi convertido de um DataFrame para um **Dicionário de Dicionários**.
- Toda a lógica de filtragem e extração de características (features) foi realizada através da iteração de dicionários e listas.
- Implementação de funções modularizadas para automação de gráficos.

---

### 🛠️ Tecnologias Utilizadas
- **Python 3**
- **Pandas** (apenas para carga inicial dos dados)
- **Matplotlib** (geração de gráficos de linha, histogramas e boxplots)
- **Google Colab**

---

### 📈 Visualizações Geradas
O projeto analisa variáveis numéricas e categóricas para identificar padrões, tais como:
- **Distribuição por Ano:** Evolução do volume de lançamentos no catálogo.
- **Análise de Frequência:** Identificação da concentração de títulos por período.
- **Identificação de Outliers:** Utilização de Boxplots para análise estatística de dispersão.
